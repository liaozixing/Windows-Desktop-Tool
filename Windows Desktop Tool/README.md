#  Windows 桌面工具 ( Windows Desktop Tool)

**版本：v1.1.1 测试版** (更新日期: 2026-01-18)

一个基于 Python 和 PyQt-Fluent-Widgets 开发的现代化 Windows 桌面实用工具集。该工具集成了网络信息查询、网速测试、文件粉碎、窗口定位工具、系统快捷工具等功能，旨在提升用户的桌面操作效率。

## 🛠️ 最近更新 (v1.1.1)
- **性能优化**：重构了文件粉碎功能的深度检查逻辑，大幅提升大批量文件拖入时的处理速度，彻底解决界面卡顿问题。
- **安全增强**：完善了系统关键文件的保护机制，支持自动识别并拦截系统核心进程占用的文件。
- **交互优化**：优化了文件粉碎界面的按钮状态控制与操作反馈。
- **本地化**：对核心 UI 组件代码进行了中文注释标注。

## 🛠️ 环境要求

- **操作系统**: Windows 10/11 (推荐 Windows 11)
- **Python 版本**: Python 3.8 或更高版本
- **依赖库**:
  - `PyQt5`: 界面框架
  - `PyQt-Fluent-Widgets`: Fluent Design 风格组件库
  - `requests`: 网络请求
  - `psutil`: 系统进程信息获取
  - `speedtest-cli`: 网络测速核心

## 🚀 快速开始

### 1. 克隆或下载项目
将项目代码下载到本地目录。

### 2. 安装依赖
打开终端（CMD 或 PowerShell），进入项目根目录，运行以下命令安装所需依赖：

```bash
pip install -r requirements.txt
```

### 3. 启动程序
在项目根目录下运行以下命令即可启动：

```bash
python main.py
```

## 💡 功能说明

### 1. 🌐 网络信息查询
- **高精度定位**：自动获取当前公网 IP 地址。
- **详尽数据**：显示地理位置（省、市）、运营商信息及数据来源。

### 2. 🚀 可视化网速测试
- **动态仪表盘**：通过仪表盘实时展示下载与上传速度。
- **实时图表**：内置测速曲线图，直观反映网络波动。
- **单位切换**：支持 Mbps 与 MB/s 单位切换。

### 3. 🧹 文件粉碎工具 (新增)
- **强制删除**：支持粉碎被占用的文件或文件夹，自动尝试解除进程占用。
- **批量处理**：支持文件/文件夹混合拖拽，支持在列表中移除不需要粉碎的项目。
- **只读破解**：自动处理只读属性，确保销毁彻底。

### 4. 🔍 窗口弹窗定位工具
- **实时识别**：拖动定位图标至任意窗口，实时显示窗口标题、句柄、PID 及所属进程路径。
- **快速追踪**：支持一键打开目标进程所在文件位置或**复制程序路径**。

### 5. 🛠️ 系统工具集成
- **快捷工具栏**：一键直达命令行、任务管理器、文件资源管理器、组策略编辑器、运行对话框及**环境变量设置**。
- **系统激活查询**：在“其他工具”中新增一键查看 Windows 系统激活状态及过期时间。

### 6. 🎨 界面与个性化
- **主题适配**：优化为默认深色模式，支持浅色切换。
- **系统托盘**：支持最小化到托盘运行，防止误关。
- **性能优化**：移除冗余废弃代码，优化启动速度与整体运行性能。
- **完全汉化**：全界面专业汉化，操作直观。
